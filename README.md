# SERM - SEO Engine Requirements Motor

## پیش‌نیازها
- Node.js 18+
- npm 9+

## نصب
\`\`\`bash
git clone https://github.com/Jimi24254/SERM.git
cd SERM
npm install
\`\`\`

## راه‌اندازی محلی
\`\`\`bash
npm run dev
\`\`\`

## استقرار
پروژه به صورت خودکار در Vercel مستقر می‌شود

## محیط‌های متغیر
- `API_KEY_GEMINI`: کلید API Gemini
- `API_KEY_GPT`: کلید API GPT
